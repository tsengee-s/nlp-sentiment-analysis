    lda_model1 = Lda(corpus=corpus1, num_topics=n, id2word=dictionary1, passes = 50)
    perplexity1 = lda_model1.log_perplexity(corpus1)
    perplexity_scores1.append(perplexity1)

    coherence_model1 = CoherenceModel(model=lda_model1, texts=sampled_df['combined_tokens'], dictionary=dictionary1, coherence='c_v')
    coherence1 = coherence_model1.get_coherence()

Comment:¶

I calculated and plotted perplexity and coherence scores for different N-values (using the CoherenceModel package) to evaluate the LDA topic model. A lower perplexity score (less negative value) means the topics are more informative. A higher coherence score suggests a better quality of topics. So, I tried choose an N that minimizes perplexity and maximizes coherence.

For the news articles, the optimal N seems to be at 10 topics (coherence peaks and perplexity is lower). 13 topics might be suitable as well, but I'm choosing to go with 10 instead, since the coherence scores are the same and perplexity only goes down a litte.

from gensim import corpora
from gensim.models import LdaModel
from gensim.parsing.preprocessing import preprocess_string

sampled_df['combined_text'] = sampled_df['clean_title'] + ' ' + sampled_df['clean_text']

# Preprocess the text data
def preprocess_text(text):
    return preprocess_string(text)

# Apply preprocessing to the 'combined' column
sampled_df['processed_text'] = sampled_df['combined_text'].apply(preprocess_text)

# Create a dictionary representation of the documents
dictionary = corpora.Dictionary(sampled_df['processed_text'])

# Create a bag-of-words corpus
corpus = [dictionary.doc2bow(text) for text in sampled_df['processed_text']]

# Train the LDA model
num_topics = 10
lda_model = LdaModel(corpus=corpus, id2word=dictionary, num_topics=num_topics, passes=10, random_state=42)

# Print the topics
topics = lda_model.print_topics(num_words=10)
for topic in topics:
    print(topic)
